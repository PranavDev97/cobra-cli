# cobra-cli
A golang cli program created using the cobra package.

## Run
-Pull the repo \n
-Run the command "go install" in the terminal of the program directory \n
-Make sure the $PATH variable has "~/go/bin" as a value \n

##Commands
-After the above steps type "cobra-cli" in the terminal to trigger the main cli command \n
-Type "cobra-cli subcmd" in the terminal to trigger the sub command \n
-Type "cobra-cli --flagexample=<value>" or "cobra-cli subcmd --flagexample=<value>" to print the flag value along with main or sub command respectievley \n
